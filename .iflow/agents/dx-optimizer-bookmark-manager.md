# Bookmark Manager DX Optimizer Configuration

## Project Overview
A React-based bookmark manager with Flask backend that helps users upload, organize, and manage their browser bookmarks.

## Current Tooling Status
- **Build Tool**: Vite with React plugin
- **Framework**: React 19 with Hooks
- **Styling**: Tailwind CSS with PostCSS
- **State Management**: Zustand
- **Routing**: React Router v7
- **HTTP Client**: Axios
- **UI Components**: Custom component library with shadcn/ui integration
- **Notifications**: Sonner
- **Icons**: Lucide React
