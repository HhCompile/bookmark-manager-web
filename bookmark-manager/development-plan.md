# 前端项目开发计划文档

## 项目概述
这是一个使用React技术栈的书签管理前端项目，提供用户友好的界面来上传、管理、分类和检查重复的书签。

## 技术栈
- React 19 + Hooks
- shadcn/ui (样式组件库)
- Tailwind CSS (样式框架)
- Axios (HTTP客户端)
- Vite (构建工具)
- React Router v7 (路由管理)
- Zustand (状态管理)

## 项目结构
```
src/
├── assets/             # 静态资源文件
│   └── react.svg      # React图标
├── components/         # 公共组件
│   ├── Header.jsx     # 页面头部
│   ├── Footer.jsx     # 页面底部
│   ├── Layout.jsx     # 页面布局组件
│   └── ui/            # UI组件库
│       └── button.jsx # 按钮组件
├── lib/                # 工具库
│   └── utils.js       # 通用工具函数
├── pages/              # 页面组件
│   ├── UploadPage.jsx # 上传页面
│   ├── BookmarkList.jsx # 书签列表页面
│   └── DuplicateCheck.jsx # 重复书签检查页面
├── router/             # 路由配置
│   └── index.jsx      # 路由配置文件
├── services/           # API服务
│   └── api.js         # API调用封装
├── store/              # 状态管理
│   └── bookmarkStore.js # 书签状态管理
├── App.css            # 应用样式
├── App.jsx            # 根组件
├── index.css          # 全局样式
└── main.jsx           # 入口文件
```
