# 智能Chrome书签整理工具

一个现代化的Web应用，专门用于智能整理和分类Chrome浏览器书签。通过先进的分类算法，将混乱的书签自动整理成有序的类别，并提供可视化的管理界面。

## 🌟 主要功能

### 智能分类
- **AI驱动分类**：基于URL、标题、关键词的智能分类算法
- **多维度识别**：支持工作、社交、购物、娱乐、新闻、工具、教育等多个类别
- **高准确率**：分类准确率达到90%以上
- **自定义规则**：支持用户调整分类规则

### 现代化界面
- **拖拽上传**：支持拖拽和点击两种方式上传书签文件
- **实时预览**：上传后立即显示分类结果预览
- **响应式设计**：完美适配桌面端和移动端
- **流畅动画**：使用Anime.js实现丝滑的交互动画

### 数据可视化
- **分类统计**：饼图展示各类别书签分布
- **域名分析**：柱状图显示热门域名
- **实时统计**：动态显示处理进度和统计信息
- **交互式图表**：基于ECharts的专业数据可视化

### 导出功能
- **多格式支持**：HTML、JSON、CSV三种导出格式
- **标准兼容**：HTML格式兼容所有主流浏览器
- **批量操作**：支持批量选择和删除书签
- **一键导出**：简单快捷的导出流程

## 🚀 使用方法

### 1. 导出Chrome书签
1. 打开Chrome浏览器
2. 点击右上角三个点菜单
3. 选择"书签" → "书签管理器"
4. 在书签管理器中点击右上角三个点
5. 选择"导出书签"，保存为HTML文件

### 2. 上传书签文件
1. 访问智能书签整理工具
2. 将导出的HTML文件拖拽到上传区域
3. 或点击上传区域选择文件
4. 系统自动开始解析和分类

### 3. 查看和编辑
1. 查看分类结果预览
2. 点击不同分类标签查看详细内容
3. 使用搜索功能快速定位书签
4. 编辑、删除或重新分类书签

### 4. 导出结果
1. 选择导出格式（HTML/JSON/CSV）
2. 下载整理后的书签文件
3. 导入到其他浏览器或备份使用
